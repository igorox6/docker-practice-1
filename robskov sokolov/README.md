# README

Данное приложение представляет собой простой REST API-сервис, реализованный с использованием **FastAPI**. Сервис предназначен для создания тем и хранения сообщений внутри этих тем. Все данные хранятся во временной памяти приложения и не сохраняются после его перезапуска.

## Описание

Приложение использует словарь `topics` для хранения тем. Каждая тема имеет уникальный идентификатор `topic_id`, который увеличивается при создании новой темы.

Каждая тема содержит:
- `id` — уникальный идентификатор темы
- `title` — название темы
- `messages` — список сообщений, относящихся к теме

Сообщение содержит:
- `author` — имя автора сообщения
- `text` — текст сообщения

Для валидации входных данных используются модели **Pydantic**:
- `Topic` — модель для создания темы (поле `title`)
- `Message` — модель для создания сообщения (поля `author`, `text`)

## API Endpoints

### POST /topics
Создаёт новую тему.

**Тело запроса**

{
  "title": "Название темы"
}
